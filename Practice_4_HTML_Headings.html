HTML Horizontal Rules

The hr tag defines a thematic break in an HTML break in an HTML page, and is most often displayed as a horizontal rule.

The hr element is used to separate content (or define a change) in an HTML page:

Example,

This is heading 1

This is some text.

This is heading 2

This is some other text.

This is heading 3

This is some other text.
